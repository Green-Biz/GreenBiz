<summary>일정관리</summary>
<div markdown="1">
1. 일정구분
- 전체일정, 전사일정, 개인일정을 구분하여 조회할 수 있다.

2. 일정 등록
- 일정 등록 시 일정 TITLE, 일수 선택 후 저장
- 관리자 아이디로 로그인시 전사 일정을 등록 할 수 있다.

3. 월, 주, 일 단위 검색
- 등록 일정의 월 단위, 주 단위, 일 단위로 조회 검색 할 수 있다.

4. 삭제 및 수정
- 일정 클릭 시 삭제 가능하며, 일정 드래그 또는 드롭다운시 일정 수정 가능.
